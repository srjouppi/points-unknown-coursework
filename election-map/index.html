Interactive maps: 2020 Presidential Election

Using Javascript, D3 and this handy PointsUnknown tutorial, I created two interactive maps of the 2020 Presidential Election.

June 3, 2022

Choropleth Map

This map uses multiple layers of polygons and data to and custom zoom levels to switch between county-level and statewide data.

Graduated Point Map

This map colors dots based on the candidate that won the county and sizes them by their margin.
